# Resonance 

Resonance is a Spotify-inspired music streaming backend built with Spring Boot.

## Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- MapStruct

## Current Features

- Spring Boot project setup
- MySQL configuration

## Planned Features

- CRUD APIs
- DTOs
- Validation
- Global Exception Handling
- JPA Relationships
- Spring Security + JWT
- File Upload & Download
- Email Integration
- Pagination & Sorting
- Dynamic Searching
- Caching
- Testing
- Docker
- Deployment
